# Free EUR Predictor

**🛢️ Early, physics-informed EUR forecasts for shale wells**

Superior day 30+ predictions. Tuned for Permian Basin (Wolfcamp / Bone Spring). Works on most unconventional plays with easy calibration.

**Completely free. No login. No installation for basic use.**
